3. text-align in CSS The text-align property in CSS is used to align text horizontally within its container. It's one of the most straightforward and essential text styling properties. Possible Values of text-align left Aligns the text to the left edge of the container. Default value for most block-level elements in left-to-right (LTR) languages like English. Example: css Copy code p { text-align: left; } right Aligns the text to the right edge of the container. Useful in right-to-left (RTL) languages or for specific designs. Example: css Copy code p { text-align: right; } center Centers the text within the container. Example: css Copy code div { text-align: center; } justify Spreads text so that each line has equal width, aligning both the left and right edges (except the last line). Often used in newspapers, books, or professional documents.
