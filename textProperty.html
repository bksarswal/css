CSS Text Properties 

1. color

Sets the color of the text.
Can use named colors, HEX, RGB, or HSL values.
p {
  color: red; /* Named color */
  color: #ff5733; /* HEX */
  color: rgb(255, 87, 51); /* RGB */
  color: hsl(12, 100%, 50%); /* HSL */
}

2. font-size

Defines the size of the text.
Units: px, em, rem, %, vw, vh.
h1 {
  font-size: 24px; /* Absolute size */
  font-size: 1.5em; /* Relative to parent */
  font-size: 150%; /* Percentage */
}

3. text-align

Aligns text horizontally.
Possible values: left, right, center, justify.
div {
  text-align: center; /* Centers text */
}

4. line-height

Controls the space between lines of text.
Can use a unit (px, %) or a multiplier (e.g., 1.5).
p {
  line-height: 1.8; /* 1.8 times the font size */
  line-height: 30px; /* Fixed height */
}

5. letter-spacing

Adjusts the space between characters.
Positive values increase space; negative values decrease it.
h2 {
  letter-spacing: 2px; /* Widens space */
  letter-spacing: -1px; /* Narrows space */
}

6. font-family

Specifies the font type for the text.
Always include a fallback font to ensure compatibility.
Example font stacks:
font-family: "Arial", sans-serif;
font-family: "Georgia", serif;
body {
  font-family: "Roboto", "Arial", sans-serif; /* Primary and fallback fonts */
}

Combining Text Properties Example


h1 {
  color: navy;
  font-size: 36px;
  text-align: center;
  line-height: 1.5;
  letter-spacing: 1px;
  font-family: "Times New Roman", serif;
}
