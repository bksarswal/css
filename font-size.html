What is font-size?

The font-size property controls how big or small the text appears on the screen.
 we can define the size using different types of units.


Units in font-size


px (Pixels):

A fixed size that doesn't change with the size of the screen or other elements.
h1 {
  font-size: 20px; /* Always 20 pixels, no matter what */
}

em (Relative to Parent):

The size is based on the font size of the element's parent.
div {
  font-size: 16px; /* Parent font size */
}

div p {
  font-size: 2em; /* 2 times the parent size: 16px × 2 = 32px */

}

rem (Relative to Root):

The size is based on the font size of the root element (usually ).
Example:
html {
  font-size: 16px; /* Root font size */
}
p {
  font-size: 1.5rem; /* 1.5 × 16px = 24px */
}

% (Percentage of Parent):

The size is a percentage of the font size of the parent.
Example:
div {
  font-size: 20px; /* Parent size */
}
div span {
  font-size: 150%; /* 150% of 20px = 30px */
}

vw and vh (Viewport Width and Height):

vw: Based on the width of the screen.
1vw = 1% of the screen width.
vh: Based on the height of the screen.
1vh = 1% of the screen height.
Example:
h1 {
  font-size: 5vw; /* 5% of the screen's width */
}

How It Works

px: Always the same size (e.g., 20px stays 20 pixels everywhere).
em: Gets bigger or smaller based on its parent element's size.
rem: Consistent scaling based on the root size (html).
%: Proportional to its parent’s size.
vw/vh:Changes depending on the screen size.
