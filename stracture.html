CSS Rule Structure

A CSS rule has two main parts:

1 Selector: 
Specifies which HTML element(s) the rule applies to.

2 Declaration Block: 
Contains one or more style declarations enclosed in {}.

 
Each declaration includes:


Property :
 The aspect of the element you want to style (e.g., color, font-size, margin).

Value: 
 The specific setting for the property (e.g., red, 16px, 10px).
Example

h1 {
  color: blue;      /* Property: color, Value: blue */
  font-size: 24px;  /* Property: font-size, Value: 24px */
}
